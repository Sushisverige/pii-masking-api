# Ethics & Responsible Use

## Human-in-the-loop
Outputs are advisory. Do not use this software as the sole basis for decisions affecting individuals (e.g., hiring, termination, access).

## Fairness / Bias
If used for evaluation (e.g., resumes), validate for bias and provide explanations. Keep decision criteria auditable.

## Prohibited Uses
- Processing data without proper authorization/consent
- Discrimination, harassment, or surveillance
- Attempting to bypass platform restrictions or terms

## Transparency
Tell users what data is processed, where it is sent/stored, and how to delete it.
